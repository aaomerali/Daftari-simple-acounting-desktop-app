# دفتري (Daftari) - Desktop POS & Accounting App

Daftari is a fully offline, secure desktop application designed for small to medium-sized retail stores with an Arabic RTL interface.

## 🌟 Key Features
- **Secure Authentication:** Built-in login with SHA-256 password hashing.
- **Unified Invoices:** Manage Sales and Purchases seamlessly with live search and filters.
- **Thermal Printing:** Optimized 80mm receipt printing for all invoice types.
- **Full Accounting:** POS, Inventory management, Expenses tracking, and Suppliers/Customers management.
- **Dynamic Settings:** Customize store details and currencies in real-time.

## 💻 Tech Stack
- Frontend: React.js (Vite) + Tailwind CSS
- Desktop: Electron.js
- Database: Local SQLite (`better-sqlite3`)

## 🚀 Getting Started

### Install
\`\`\`bash
npm install
\`\`\`

### Run locally
\`\`\`bash
npm run dev
\`\`\`

### Build (Windows)
\`\`\`bash
npm run build:win
\`\`\`

*Note: The default Admin login is `admin@daftari.com` / `daftari123`.*
